HTML stands for Hyper Text Markup Language and is used to create websites (like this one!). It uses tags to add content to the website. For example, an HTML document can look like this:

<!doctype html>
<html>
    <head>

        <title>Hello World</title>

    </head>

    <body>
        <p>Hello world!<\p>
    </body>

</html>

Structure

If you look at the structure of the code you can see <html> and </html> at the beginning and at the end. These are known as tags. <HTML-tag> is called an opening tag and </HTML-tag> is known as the closing tag. You will see several tags that do not have closing tags; they are called self-closing tags. For example:
        <img src="link" />
This tag inserts an image to the document. There is no opening or closing tags; it is self-closing. Usually, self-closing tags end with a />.

Doctype

Include <!doctype html> at the top of every html document. It is to let the browser know that it is an html document.

Title Tag

The <title>Hello World</title> tag will display the surrounded text on the tab of the website.

Paragraph and break tags

To display a block of text, we use the paragraph tag. <p>text goes here</p> will display the surrounded text. Something you will notice is that html ignores whitespace. What this means is that adding spaces inbetween words will not change the display.
For example, <p>text   goes      here</p> will look the same as <p>text goes here</p>. Actually there is a way, but we will learn that when we get to (add link here)entities.
However, if you want to skip a line, we use the break tag: <br/>. Notice that the break tag is a self-closing tag.
<p>text goes <br/>here</p> will display:

text goes
here

Horizontal Line

You probably notice the horizontal lines throughout the page. To insert one, use the self-closing tag: <hr/>.

Bold, Italic, Underline, Strikethrough

Surround text between <strong></strong> tags to bold the text. Surround between <em></em> (emphasize) tags to italicize. Surround between <u></u> tags to underline. Surround between <strike></strike> tags for strikethrough.
For example, the following code:
<strong>This is a bold piece of text. </strong>This word will be in <em>italics</em>. I want this <u>word</u> to be underlined. <strike>This line is wrong.</strike> will display:

This is a bold piece of text. This word will be in italics. I want this word to be underlined. This line is wrong.

Comments

Usually, you would like to leave notes throughout your html document. They are useful for recalling topics, what the code does, etc.
To insert a comment: <!--Comment-->. You will know if you typed it right if it greys out or if something similar occurs.

Ordered Lists and Unordered Lists

If you want to simply list several items you have a choice between an ordered list and an unordered list. An ordered list has a structure that tells you what is first, second, etc.
The syntax for an ordered list is the tag <ol></ol> and the items will be between the tags. To insert an item, you use the tag <li></li>.For example:

<p><strong>Ordered List</strong></p>
<ol>
   <li>First</li>
   <li>Second</li>
   <li>Third</li>
</ol>

Will produce an ordered list:

Ordered List

  1. First
  2. Second
  3. Third

For an unordered list, there is no structure; we do not know what is first, second, etc. The syntax is very similiar to that of an ordered list, but we use the tag <ul></ul>.
For example:
<p><strong>My Favorite Foods</strong></p>
<ul>
   <li>Burgers</li>
   <li>Chocolate</li>
   <li>Sushi</li>
</ul>

Will produce:

My Favorite Foods

  • Burgers
  • Chocolate
  • Sushi

Images

To insert a downloaded image, first make sure the image is in the same directory as your html document. The syntax to insert an image is: <img src="image-name.file-extension (jpeg, png, etc)/>. Note that it is a self-closing tag.
For example, using an image I got off of Google: <img src="google-newyears.gif/> will display:

You might notice that the image is huge. To change the width/height of it, add the width and/or height parameter(s).
For example:
<img src="google-newyears.gif" width="100"/> will display:

You will notice again that if you only put one parameter, the image will proportionately change size. Obviously, if you add both parameters, the image will change to those dimensions.
You can also add images via an url.
For example, using an image found on Wikipedia:
<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/5/5a/1_epcot_illuminations_2010.jpg/1024px-1_epcot_illuminations_2010.jpg" width="400"/>

Forms

You have definitely seen forms on websites before. Examples are typing your username or password, writing a paragraph in a textbox, selecting options from a list, etc. To insert a form, the syntax is: <form></form>. Every type of form will be inbetween the form tags.
The first type of form we will look at will be the one-line textbox.
The syntax is: <input type="text"/>.

If you want the box to have some text initially, we have to use the placeholder parameter. If we type <input type="text" placeholder="type here..."/> it will get us:

We can change the type of input it is by changing the type. If we change the type to password, the text will be censored. If we change the type to email, the text must be in email format.



To create a textbox, we change the input to textarea. For example:
<textarea>My content goes here</textarea> will display:

To create a drop-down form, we change the input to select. Between the select tags will be the options. For example:
<select>
    <option>Select an option...</option>
    <option>1</option>
    <option>2</option>
    <option>3</option>
</select>

<input type="radio" name="color" value="red"/> Red
<input type="radio" name="color" value="green"/> Green
Will give us:

To create a checkbox form, we change the type to checkbox. For example:
<input type="checkbox"/> Dog
<input type="checkbox"/> Cat
<input type="checkbox"/> Fish

This will give us:

<input type="submit"/>

If you want the button to say something different, we have to use the value parameter. For example:

Links

If we want to insert a link to another website or html page, we use the <a href="link">Hyperlink Text</a>.
For example, if we want to link to Google:
<a href="http://google.com">Link to Google<a/>

Link to Google

Note that you need the http://. If you want to create a link to another html document, replace the link with the name of the html document with the extension.
Something else useful with links is helping the user navigate the webpage. An example is the table of contents at the top of this page. We have to have an anchor tag. For example, let us say we want a link to get back to the top of the page. Towards the top of the page, we leave an anchor tag:
<a name="top"/>
Note that it is a self-closing tag. Now you leave the link somewhere:
<a href="#top">Link to Top</a>
Note the # before the name!

Tables

Tables are used to insert several rows and columns of data. To create a table, we use the <table></table> tags. Now we want to insert a row. To do this we use the <tr></tr> tags, where tr stands for tr. Finally, for each row we want columns of data. We have two options here: table headers, which makes the text bold, or regular table data. We use <th></th> for the table headers and <td></td> for the table data.
For example, the complete code for a table:

<table>
    <tr>     <!--tr represents a table row-->
        <th>Names</th>     <!--th is a table header (bolder)-->
        <th>Age</th>
        <th>Gender</th>
    </tr>
    <tr>
        <td>Calvin</td>     <!--td is a table column-->
        <td>19</td>
        <td>Male</td>
    </tr>
    <tr>
        <td>Iris</td>
        <td>19</td>
        <td>Female</td>
    </tr>
</table>

Which will give us:

Names Age Gender
Calvin 19 Male
Iris 19 Female

Note, that there is no structure for a table! One of those rows can have two or four columns even though three table headers are created.
